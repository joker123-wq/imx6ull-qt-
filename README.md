# IMX6ULL Qt Multimedia System

基于 i.MX6ULL + Qt 开发的综合多媒体项目，包含图片浏览、视频播放、音乐播放、开机界面、亮度调节、文件自动检索等功能。

## 功能介绍
- 开机启动界面
- 图片自动扫描与浏览
- 视频播放
- 音频播放
- 文件自动检索
- 亮度调节
- 类手机风格UI

## 开发环境
- 开发板：i.MX6ULL
- 系统：Linux
- 框架：Qt 5.5.1
- 显示方式：linuxfb

## 项目结构
- `src/` 核心源码
- `include/` 头文件
- `res/` 资源文件
- `README.md` 项目说明

## 项目亮点
1. 基于嵌入式 Linux 平台实现 Qt 图形界面
2. 支持多媒体资源自动检索
3. 完成启动页、主界面、播放器等模块设计
4. 适合作为嵌入式 GUI 项目展示
